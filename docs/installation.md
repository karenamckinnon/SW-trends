# Installation

## Stable release

To install SW trends, run this command in your terminal:

```sh
uv add SW-trends
```

Or if you prefer to use `pip`:

```sh
pip install SW-trends
```

## From source

The source files for SW trends can be downloaded from the [Github repo](https://github.com/karenamckinnon/SW_trends).

You can either clone the public repository:

```sh
git clone git://github.com/karenamckinnon/SW_trends
```

Or download the [tarball](https://github.com/karenamckinnon/SW_trends/tarball/master):

```sh
curl -OJL https://github.com/karenamckinnon/SW_trends/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd SW_trends
uv pip install .
```
